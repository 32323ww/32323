# 毛概刷题站

这是毛概刷题网站的部署仓库。

## Cloudflare Pages 设置

- Framework preset: `None`
- Build command: `echo no build`
- Build output directory: `quiz_site`

## Netlify 设置

仓库中也保留了 `netlify.toml` 和 `netlify/functions/`，如果继续使用 Netlify，可以直接按现有配置部署。

## 注意

不要把豆包 API Key、CloudBase `AUTH_SECRET`、本地运行日志、PDF/Word 资料上传到公开仓库。
